Quick Start:
-----------
1. Get Bot Token from @BotFather on Telegram
2. Copy .env.example to .env and configure:
   - BOT_TOKEN=your_token_here
   - ADMIN_IDS=your_telegram_id

3. Start services:
   docker-compose up -d

4. Seed questions:
   docker-compose exec bot python scripts/seed_questions.py

5. Open Telegram, find your bot, send /start

Game Flow:
----------
1. Player 1 creates room → gets 6-character code
2. Player 2 joins using the code
3. 3-second countdown
4. 10 questions, 15 seconds each
5. Answer with inline buttons
6. Scores calculated with time bonus
7. Final results with statistics
8. ELO ratings updated
9. Match saved to history

Scoring System:
--------------
• Correct Answer: 100 points + time bonus (up to 7.5)
• Incorrect Answer: 0 points
• No Answer: 0 points
• Time Bonus: (15 - response_time) * 0.5

ELO Rating:
----------
• Starting Rating: 1000
• K-Factor: 32
• Win vs Higher Rated: +32 to +50
• Win vs Lower Rated: +10 to +25
• Loss: Inverse points (negative)
• Draw: No change
